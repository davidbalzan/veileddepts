## Obsolete Documentation

The `obsolete/` folder contains outdated documentation that's no longer relevant but kept for historical reference. These files document approaches that were tried but ultimately replaced with better solutions.

## Contributing to Documentation

When adding new documentation:
1. Place current/active docs in the main `docs/` folder
2. Move completed feature docs to `archive/` when the feature is finished
3. Move outdated approaches to `obsolete/` when they're no longer relevant
4. Update this index when adding new files
